About CSS

CSS (Cascading Style Sheets) is the language used to style HTML documents. It describes how elements should be displayed on screen, on paper, or in other media. Its the skin on top of the html skeleton.

CSS Libraries

CSS libraries help you avoid copy pasting the code 😅

  • Bootstrap
  • Tailwind CSS – Utility-first, build anything with atomic classes! (Recommended)
  • Material UI – Google's design magic for React wizards ✨
  • Foundation – The Swiss Army knife of responsive design 🛠️
  • Semantic UI – Style your site with readable, human-friendly code 😎
  • Bulma – Modern CSS with Flexbox superpowers 💪
  • Pure CSS – Tiny, minimal, and just the essentials for the purists 🧼
